# تنظیم متغیرهای محیطی فرانت‌اند برای لیارا

## متغیرهای محیطی مورد نیاز

برای اتصال فرانت‌اند به بک‌اند، باید متغیر محیطی زیر را در **تنظیمات برنامه لیارا** (mcl-front) اضافه کنید:

### برای پروداکشن (لیارا):
```
REACT_APP_API_BASE_URL=https://mcl-backend.liara.run
```

### برای محیط لوکال (اختیاری):
اگر می‌خواهید در محیط لوکال کار کنید، یک فایل `.env` در پوشه `front` ایجاد کنید:

```
REACT_APP_API_BASE_URL=http://localhost:8081
```

## مراحل تنظیم در لیارا:

1. وارد **کنسول لیارا** شوید
2. برنامه **mcl-front** را انتخاب کنید
3. به بخش **تنظیمات** بروید
4. روی **متغیرهای محیطی** کلیک کنید
5. متغیر جدید اضافه کنید:
   - **نام**: `REACT_APP_API_BASE_URL`
   - **مقدار**: `https://mcl-backend.liara.run`
6. روی **ذخیره** کلیک کنید
7. برنامه را **ری‌استارت** کنید

⚠️ **نکته مهم**: اگر نام برنامه بک‌اند شما در لیارا متفاوت است (مثلاً `my-backend`)، آدرس را به `https://my-backend.liara.run` تغییر دهید.

## بررسی اتصال:

بعد از ری‌استارت:
1. برنامه را در مرورگر باز کنید
2. Developer Console (F12) را باز کنید
3. به تب **Network** بروید
4. سعی کنید لاگین کنید یا عملیاتی انجام دهید
5. باید ببینید که درخواست‌ها به `https://mcl-backend.liara.run` ارسال می‌شوند
